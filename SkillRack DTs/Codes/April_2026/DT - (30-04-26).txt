Problem Statement :-

Print All Elements From Mth Integer Circularly in DT (Daily Test) in SkillRack in Python - Programming

Code :-

N , M = map(int , input().split())

values_list = list(map(int , input().split()))

values_list_index = ((M % N) if((M % N) == 0) else ((M % N) - 1)) if(M > N) else (M - 1)

for count in range(1 , (N + 1)):
    
    print(values_list[values_list_index] , end = " ")
    
    values_list_index = ((values_list_index + 1) % N)
