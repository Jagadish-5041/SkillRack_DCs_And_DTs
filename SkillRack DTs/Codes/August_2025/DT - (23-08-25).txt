Problem Statement :-

Vertical Binary Representation in DT (Daily Test) in SkillRack in Python Programming

Code :-

N = int(input())

binary_strings_list = [bin(number)[2 : ] for number in range(N , 0 , -1)]

R = len(max(binary_strings_list , key = len))

C = N

matrix = [["" for j in range(0 , C)] for i in range(0 , R)]

for j in range(0 , C):
    
    for i in range(0 , len(binary_strings_list[j])):
        
        matrix[i][j] = binary_strings_list[j][i]

[print(*row_list , sep = "") for row_list in matrix]
