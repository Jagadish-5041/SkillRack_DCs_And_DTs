Problem Statement :-

Odd / Even Rows Pattern in DC (Daily Challenge) in SkillRack in Python - Programming

Code :-

N = int(input())

value = 1

for i in range(1 , (N + 1)):
    
    value = (value - 1) if(value > 1) else (value)
    
    for j in range(1 , (i + 1)):
        
        print(value , end = " ")
        
        value += 2
    
    print()
