Function - printEveryRowCharacters() in DC (Daily Challenge) in SkillRack in C - Programming

Code :-

void printEveryRowCharacters(int R , int C , char *matrix){
    
    for(int i = 0 ; (i < R) ; i++){
        
        for(int j = i ; (j < C) ; j += (i + 1)){
            
            printf("%c " , matrix[((i * C) + j)]);
        }
    }
}
