Problem Statement :-

Flowchart Logic 053 in DC (Daily Challenge) in SkillRack in Python - Programming

Code :-

X , Y = map(int , input().split())

flag = True

while(flag):
    
    K = int(input())
    
    for ctr in range(1 , (K + 1)):
        
        X = (X + 1) if((ctr % 2) == (K % 2)) else (X)
        
        Y = (Y) if((ctr % 2) == (K % 2)) else (Y - 1)
        
        print(X , Y)
    
    flag = (X < Y)
